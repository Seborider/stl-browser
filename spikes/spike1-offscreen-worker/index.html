Spike 1 — OffscreenCanvas + three.js Worker

Renders STL fixtures to 512×512 PNGs inside a single Web Worker using OffscreenCanvas. Targets: 100k tris < 300 ms, 1M tris < 2000 ms (worker-side, message-in → bytes-out).

Generate fixtures first: node gen-stl.mjs, then serve this directory over HTTP and open in Safari.
